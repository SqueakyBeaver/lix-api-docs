Detailed Description

Mix-in class for implementing permanent roots as a pair of a direct (strong) reference and indirect weak reference to the first reference.

See methods for details on the operations it represents.

◆ addPermRoot()

kj::Promise< Result< Path > > nix::IndirectRootStore::addPermRoot ( const StorePath & storePath,
const Path & gcRoot )
finaloverridevirtual

Implementation of LocalFSStore::addPermRoot where the permanent root is a pair of

  • The user-facing symlink which all implementations must create
  • An additional weak reference known as the "indirect root" that points to that symlink.

The garbage collector will automatically remove the indirect root when it finds that the symlink has disappeared.

The implementation of this method is concrete, but it delegates to addIndirectRoot() which is abstract.

Implements nix::LocalFSStore.
